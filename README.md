# ベスト10実績 ＆ 今月売上実績突合ツール

本ツールは、基準となる部門別ベスト10商品の売上データに対し、最新の単品別販売管理表から今月の実績値（売上高、比較日比）を自動で抽出して突合し、Excel形式のレポートを生成するツールです。

## フォルダ構成

```text
Best10_analysis/
├── .python-version               # Pythonバージョン指定 (3.12)
├── pyproject.toml                # プロジェクト設定・依存関係管理
├── uv.lock                       # 依存関係のロックファイル
├── run_analysis.bat              # ダブルクリックで自動実行するバッチファイル
├── src/
│   └── main.py                   # 処理実行スクリプト本体
├── input_data/                   # 参照Excelデータを配置するフォルダ
│   └── .gitkeep
└── output_data/                  # 結果Excelファイルが出力されるフォルダ
    └── .gitkeep
```

## 使用方法・実行手順

### 1. データの配置
**`input_data/`** フォルダ内に、以下の2種類のExcelファイルを配置します。

* **基準ベスト10データ**: `ベスト１０実績_*.xlsx` (例: `ベスト１０実績_202506.xlsx`)
* **最新販売管理表**: `販売管理表(単品別売上実績)-*.xlsx` (例: `販売管理表(単品別売上実績)-202606111652.xlsx`)

> **【日付の自動検出について】**
> ファイル名に含まれる日付やタイムスタンプ部分は毎回変わっても問題ありません。フォルダ内に複数ファイルが存在する場合、更新日時が最も新しいファイルを自動的に検出して読み込みます。

### 2. スクリプトの実行
フォルダ直下にある **`run_analysis.bat`** をダブルクリックして実行します。
実行するとコンソール上で「参照する店舗名」の入力を求められます（例：`全店`、`王子店`、`芦屋店` など。何も入力せずにエンターキーを押すとデフォルトの `全店` が選択されます）。
指定された店舗名のシートを基準データとして読み込み、分析を実行します。

### 3. 結果の確認
実行完了後、**`output_data/`** フォルダ内に以下のレポートファイルが生成・更新されます。

* **`best10_with_current_sales_[店舗名].xlsx`**: 指定した店舗の基準ベスト10商品に対し、今月の「売上実績」および「比較日比」を突合した最新データレポート（部門ごとにシートが分かれた構成）。

## 開発・実行環境

* **Python**: 3.12 以上
* **パッケージマネージャー**: `uv`
* **依存ライブラリ**: `pandas`, `openpyxl`
